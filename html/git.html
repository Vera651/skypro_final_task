Git

  • git init — создаёт новый локальный репозиторий в выбранной папке.
  • git clone — скачивает удалённый репозиторий и создаёт его локальную копию.
  • git status — отображает актуальное состояние файлов в рабочем каталоге и индексе, включая изменённые файлы, добавленные, находящиеся вне отслеживания и прочие изменения.
  • git add — добавляет файлы в индекс (подготавливает к коммиту).
  • git branch — отображает перечень всех локальных веток проекта.
  • git checkout <ветка> — осуществляет переход на указанную ветку.
  • git push — передаёт сделанные локально изменения в связанный удалённый репозиторий для их сохранения и совместной работы.
  • git commit -m «Описание изменений» — формирует новый коммит с комментарием, описывающим суть внесённых правок.
